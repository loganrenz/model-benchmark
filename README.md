# Project Viewer (Nuxt 4 + Nuxt UI v4)

Mobile-first Project Viewer with a bottom “Project Explorer” drawer powered by `UTree`.

## Data source

- `public/data/manifest.json` (single source of truth)

## Dev

```bash
npm install
npm run dev
```

## Lint / Build

```bash
npm run lint
npm run build
```
